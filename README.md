# Expense Tracker App

## Description
A mobile application for tracking expenses and managing personal finances.

## Features
- User authentication (Login, Sign-up)
- Expense and income tracking
- Budget management
- Flashcards for financial education
- Transaction history
- Data persistence using SQLite

## Technologies Used
- Android SDK
- SQLite (for local data storage)
- Java
- AndroidX Libraries (appcompat, constraintlayout, lifecycle, navigation)
- Material Design
- RecyclerView
- CardView
- MPAndroidChart (for data visualization)
- jBCrypt (for password hashing)
- Retrofit (for networking)
- GSON (for JSON parsing)
- OkHttp Logging Interceptor

## Setup
To set up the project locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/BudgetTracker.git
    cd BudgetTracker
    ```
2.  **Open in Android Studio:**
    Open the project in Android Studio. Android Studio will automatically download the necessary Gradle dependencies.
3.  **Run the application:**
    Connect an Android device or use an emulator to run the application.
